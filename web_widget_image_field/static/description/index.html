This module add a new widget to add an image on the right of a relation field. It like the many2one_avatar widget but for all relation field. In the relation field, you must inherit the 'image.mixin' model.

Example:

<field name="track_id" widget="many2one_image" options="{'size': 'image_256'}"/>
